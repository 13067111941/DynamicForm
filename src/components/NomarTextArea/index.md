## API

|参数|说明|类型|默认值|是否必填|
|--|--|--|--|--|
|type|表单类型|string|''|是|
|title|标题|string|''|是|
|fieldProps|文本属性|string|''|是|
|placeholder|placeholder|string|''|否|
|required|必填判断|boolean|false|否|
|rows|行数|number|3|否|
|editable|可编辑判断|boolean|false|true|
